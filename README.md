<h1>Dashboard de Fluxo de Caixa</h1>

<h2>Descrição</h2>
<p>Este projeto consiste em um Dashboard de Fluxo de Caixa desenvolvido no Power BI. Ele tem como objetivo facilitar a visualização e análise das movimentações financeiras de uma empresa, oferecendo insights sobre receitas, despesas, saldo disponível e projeções futuras.</p>

<h2>Funcionalidades</h2>
<ul>
    <li><strong>Monitoramento Financeiro</strong>: Acompanhamento detalhado das entradas e saídas de caixa.</li>
    <li><strong>Gráficos Interativos</strong>: Representação visual dos dados financeiros para melhor análise.</li>
    <li><strong>Análise Temporal</strong>: Comparação de desempenho financeiro ao longo do tempo.</li>
    <li><strong>Segmentação de Dados</strong>: Filtros dinâmicos para análise por período, categoria e origem dos valores.</li>
    <li><strong>Projeções e Tendências</strong>: Estimativa de saldo futuro com base no histórico financeiro.</li>
</ul>

<h2>Tecnologias Utilizadas</h2>
<ul>
    <li><strong>Power BI</strong>: Ferramenta principal para a criação do dashboard e visualização dos dados.</li>
    <li><strong>DAX (Data Analysis Expressions)</strong>: Utilizado para criação de métricas e cálculos personalizados.</li>
    <li><strong>Power Query</strong>: Responsável pela transformação e modelagem dos dados.</li>
    <li><strong>Fontes de Dados</strong>: Planilhas Excel, bancos de dados SQL ou arquivos CSV, conforme necessidade do projeto.</li>
</ul>

<h2>Como Utilizar</h2>
<ol>
    <li><strong>Baixe o arquivo</strong>: O dashboard está no formato <code>.pbix</code>, pronto para ser aberto no Power BI Desktop.</li>
    <li><strong>Abra no Power BI</strong>: Utilize o Power BI Desktop para visualizar e interagir com os gráficos.</li>
    <li><strong>Atualize os dados</strong>: Caso precise atualizar as informações, importe novas planilhas ou conecte-se à fonte de dados desejada.</li>
    <li><strong>Personalização</strong>: Os filtros interativos permitem segmentar informações conforme sua necessidade.</li>
</ol>

<h2>Requisitos</h2>
<ul>
    <li><strong>Power BI Desktop</strong> instalado.</li>
    <li>Fonte de dados conectada ou atualizada conforme necessidade.</li>
</ul>
